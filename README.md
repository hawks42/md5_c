# md5_c

Ce programme calcule le **hash MD5** d'une chaîne de caractères, basé sur la spécification de la **RFC 1321**.

## Utilisation

```bash
./md5_program "string"
```

##Exemple

```bash
./md5_program "azerty"
> MD5 hash : ab4f63f9ac65152575886860dde480a1
```
